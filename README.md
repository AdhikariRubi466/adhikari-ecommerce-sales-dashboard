# 📊 Adhikari Ecommerce Sales Dashboard

A simple and interactive **Power BI dashboard** for analyzing ecommerce sales data.  
It helps understand sales, profit, customer behavior, and product performance in a clear visual way.

---

## 🚀 What this project shows

- Total Sales, Profit, Quantity, and AOV  
- Profit by product sub-category  
- Monthly profit trends  
- Customer-wise sales contribution  
- Payment mode distribution  
- Filters for Quarter and State  

---

## 🛠️ Built With

- Power BI  
- Power Query  



This project is built for learning and demonstrating **data analytics and visualization skills**.
